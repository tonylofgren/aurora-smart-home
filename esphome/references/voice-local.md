# Local Voice Assistant

Guide för att bygga en helt lokal röstassistent med ESPHome och Home Assistant.

## Översikt

En lokal röstassistent kräver inga molntjänster och fungerar även utan internet. Alla komponenter körs lokalt:

- **Wake Word** - Micro Wake Word (ESP32-S3) eller openWakeWord (HA)
- **Speech-to-Text** - Whisper eller faster-whisper
- **Intent Recognition** - Home Assistant Assist
- **Text-to-Speech** - Piper

## Hårdvarukrav

### Minsta krav (Wake Word på HA)

| Komponent | Krav |
|-----------|------|
| ESP32 | Valfri variant |
| Mikrofon | I2S INMP441 eller PDM |
| Högtalare | I2S MAX98357A |

### Rekommenderat (Wake Word på enhet)

| Komponent | Krav |
|-----------|------|
| ESP32-S3 | PSRAM krävs |
| Mikrofon | I2S INMP441 (bäst) |
| Högtalare | I2S MAX98357A + 3W speaker |
| Extra | AMP enable pin för tysta pauser |

## Kopplingsschema

### INMP441 Mikrofon (I2S)

```
INMP441         ESP32-S3
--------        --------
VDD     ───────  3.3V
GND     ───────  GND
SD      ───────  GPIO16 (data)
WS      ───────  GPIO17 (word select/LRCLK)
SCK     ───────  GPIO18 (clock)
L/R     ───────  GND (vänster kanal)
```

### MAX98357A Förstärkare (I2S)

```
MAX98357A       ESP32-S3
---------       --------
VIN     ───────  5V
GND     ───────  GND
DIN     ───────  GPIO15 (data)
BCLK    ───────  GPIO14 (bit clock)
LRC     ───────  GPIO13 (left/right clock)
SD      ───────  GPIO48 (enable, optional)
GAIN    ───────  (leave unconnected = 9dB)
```

## Minimal Voice Assistant

Wake word detekteras av Home Assistant:

```yaml
# Generated by esphome@aurora-smart-home v1.0.0
# https://github.com/tonylofgren/aurora-smart-home

esphome:
  name: voice-assistant-basic
  friendly_name: "Röstassistent"
  platformio_options:
    board_build.flash_mode: dio

esp32:
  board: esp32-s3-devkitc-1
  framework:
    type: esp-idf

psram:
  mode: octal
  speed: 80MHz

wifi:
  ssid: !secret wifi_ssid
  password: !secret wifi_password

api:

ota:
  platform: esphome

logger:

# I2S Mikrofon
i2s_audio:
  - id: i2s_in
    i2s_lrclk_pin: GPIO17
    i2s_bclk_pin: GPIO18

  - id: i2s_out
    i2s_lrclk_pin: GPIO13
    i2s_bclk_pin: GPIO14

microphone:
  - platform: i2s_audio
    id: mic
    i2s_audio_id: i2s_in
    adc_type: external
    i2s_din_pin: GPIO16
    pdm: false
    bits_per_sample: 32bit
    channel: left

speaker:
  - platform: i2s_audio
    id: speaker
    i2s_audio_id: i2s_out
    dac_type: external
    i2s_dout_pin: GPIO15
    mode: mono

voice_assistant:
  id: va
  microphone: mic
  speaker: speaker
  use_wake_word: false  # Wake word på HA
  noise_suppression_level: 2
  auto_gain: 31dBFS
  volume_multiplier: 2.0

  on_listening:
    - light.turn_on:
        id: led
        effect: "Listening"

  on_stt_end:
    - light.turn_on:
        id: led
        effect: "Processing"

  on_tts_start:
    - light.turn_on:
        id: led
        effect: "Speaking"

  on_end:
    - light.turn_off: led
    - wait_until:
        not:
          speaker.is_playing:
    - voice_assistant.start:  # Auto-restart

  on_error:
    - light.turn_on:
        id: led
        red: 100%
        green: 0%
        blue: 0%
    - delay: 1s
    - light.turn_off: led

# Status LED
light:
  - platform: esp32_rmt_led_strip
    id: led
    pin: GPIO48
    num_leds: 1
    rmt_channel: 0
    chipset: SK6812
    rgb_order: GRB
    effects:
      - pulse:
          name: "Listening"
          transition_length: 250ms
          update_interval: 250ms
      - pulse:
          name: "Processing"
          transition_length: 100ms
          update_interval: 100ms
      - pulse:
          name: "Speaking"
          transition_length: 500ms
          update_interval: 500ms
```

## Med Micro Wake Word (On-Device)

Wake word körs direkt på ESP32-S3:

```yaml
# Generated by esphome@aurora-smart-home v1.0.0
# https://github.com/tonylofgren/aurora-smart-home

esphome:
  name: voice-assistant-local
  friendly_name: "Lokal Röstassistent"
  platformio_options:
    board_build.flash_mode: dio
    board_build.psram_type: opi

esp32:
  board: esp32-s3-devkitc-1
  flash_size: 16MB
  framework:
    type: esp-idf
    sdkconfig_options:
      CONFIG_ESP32S3_DEFAULT_CPU_FREQ_240: "y"
      CONFIG_ESP32S3_DATA_CACHE_64KB: "y"
      CONFIG_ESP32S3_DATA_CACHE_LINE_64B: "y"

psram:
  mode: octal
  speed: 80MHz

wifi:
  ssid: !secret wifi_ssid
  password: !secret wifi_password

api:
  on_client_connected:
    - voice_assistant.start_continuous:

ota:
  platform: esphome

logger:
  level: DEBUG

i2s_audio:
  - id: i2s_in
    i2s_lrclk_pin: GPIO17
    i2s_bclk_pin: GPIO18
  - id: i2s_out
    i2s_lrclk_pin: GPIO13
    i2s_bclk_pin: GPIO14

microphone:
  - platform: i2s_audio
    id: mic
    i2s_audio_id: i2s_in
    adc_type: external
    i2s_din_pin: GPIO16
    pdm: false
    bits_per_sample: 32bit
    channel: left

speaker:
  - platform: i2s_audio
    id: speaker
    i2s_audio_id: i2s_out
    dac_type: external
    i2s_dout_pin: GPIO15

# Micro Wake Word - körs på ESP32-S3
micro_wake_word:
  models:
    - model: hey_jarvis  # Inbyggd modell
  on_wake_word_detected:
    - voice_assistant.start:
        wake_word: !lambda return wake_word;

voice_assistant:
  id: va
  microphone: mic
  speaker: speaker
  use_wake_word: true
  noise_suppression_level: 2
  auto_gain: 31dBFS
  volume_multiplier: 2.0

  on_listening:
    - light.turn_on:
        id: led
        red: 0%
        green: 0%
        blue: 100%

  on_stt_end:
    - light.turn_on:
        id: led
        red: 100%
        green: 100%
        blue: 0%

  on_tts_start:
    - light.turn_on:
        id: led
        red: 0%
        green: 100%
        blue: 0%

  on_end:
    - light.turn_off: led
    - micro_wake_word.start:  # Lyssna efter wake word igen

  on_error:
    - light.turn_on:
        id: led
        red: 100%
        green: 0%
        blue: 0%
    - delay: 1s
    - micro_wake_word.start:

light:
  - platform: esp32_rmt_led_strip
    id: led
    pin: GPIO48
    num_leds: 1
    rmt_channel: 0
    chipset: WS2812
    rgb_order: GRB
```

## Home Assistant Setup

### 1. Installera Whisper

```yaml
# HA Add-on: Whisper
# Eller i configuration.yaml:
whisper:
```

**Rekommenderade modeller:**
- `tiny.en` - Snabbast, engelska
- `base` - Balanserad
- `small` - Bättre accuracy

### 2. Installera Piper

```yaml
# HA Add-on: Piper
# Röster för svenska:
# - sv_SE-nst-medium
```

### 3. Konfigurera Assist Pipeline

Settings → Voice assistants → Add assistant:
- **Conversation agent:** Home Assistant
- **Speech-to-text:** Whisper
- **Text-to-speech:** Piper
- **Wake word:** (micro wake word om på enhet)

## Med Fysisk Knapp

Starta röstassistent med knapptryckning:

```yaml
binary_sensor:
  - platform: gpio
    pin:
      number: GPIO0
      mode: INPUT_PULLUP
      inverted: true
    id: button
    on_press:
      - voice_assistant.start:
    on_release:
      - voice_assistant.stop:
```

## Felsökning

### Mikrofonen fungerar inte

1. **Kontrollera kablar** - I2S är känslig för dåliga anslutningar
2. **Kontrollera pinnar** - LRCLK, BCLK, DIN måste matcha
3. **Testa med logger:**
```yaml
logger:
  level: VERBOSE
```

### Wake word detekteras inte

1. **PSRAM krävs** - Micro Wake Word behöver ESP32-S3 med PSRAM
2. **Modell saknas** - Ladda ner och specifiera modell
3. **Känslighet** - Öka `vad_threshold`:
```yaml
micro_wake_word:
  vad_threshold: 1
```

### Dålig ljudkvalitet

1. **Förstärkning:**
```yaml
voice_assistant:
  auto_gain: 31dBFS
  noise_suppression_level: 2
  volume_multiplier: 3.0
```

2. **Placering** - Undvik eko och bakgrundsljud
3. **Högtalare** - Använd minst 3W högtalare

### Lång latens

1. **Whisper modell** - Använd `tiny.en` för snabbast
2. **Lokal server** - Kör Whisper på kraftfull maskin
3. **WiFi** - Använd 5GHz eller statisk IP
