# Smart Thermostat Relay Example

> *Generated by [aurora@aurora-smart-home (esphome skill)](https://github.com/tonylofgren/aurora-smart-home)*

A smart room thermostat that drives a relay for a gas boiler or electric heater. Uses ESPHome's built-in `climate.thermostat` platform with hysteresis, anti short-cycle protection, and a hard safety cutoff.

## Features

- **Accurate Room Sensing**: DS18B20 1-Wire probe on a wire (away from ESP32 self-heating)
- **Hysteresis Control**: 0.5 C deadband each side of setpoint (1.0 C total)
- **Anti Short-Cycle**: Minimum 15 min off-time between boiler calls
- **Comfort / Eco Presets**: One-press switching between 21 C and 17 C
- **Manual Override**: Boiler relay exposed as a switch entity for testing
- **Safety Cutoff**: Hard relay shutoff if room exceeds 28 C

## Hardware List

### ESP32 Controller
- ESP32 DevKit (or ESP32-C3 Mini for a smaller install)

### Sensors and Actuators
| Component | Purpose | Interface |
|-----------|---------|-----------|
| DS18B20 probe | Room temperature | 1-Wire (GPIO + 4.7k pullup) |
| SSR or relay module | Boiler call-for-heat | GPIO |

### Wiring

```
ESP32 DevKit
├── GPIO4 ──── DS18B20 data (with 4.7k resistor to 3.3V)
├── GPIO5 ──── Relay IN (SSR or opto-isolated mechanical)
├── 3.3V ──── DS18B20 VCC
├── 5V  ──── Relay VCC (most modules need 5V)
└── GND  ──── Common ground (probe, relay, ESP32)
```

> **Important**: Wire the relay contacts in parallel with the existing thermostat terminals on your boiler. Most boilers expect a dry contact closure (24V or low-voltage signal) on the "call for heat" / "thermostat" input. Check your boiler manual before connecting.

## Installation

1. **Generate secrets**:
   Copy `secrets.yaml.example` to `secrets.yaml` and fill in WiFi, API key, and OTA password.

2. **Flash the device**:
   ```bash
   esphome run smart-thermostat-relay.yaml
   ```

3. **Mount the probe**:
   Place the DS18B20 at chest height, away from windows, direct sun, and heat sources. Run the probe wire at least 30 cm from the ESP32 board so the board's own heat does not bias the reading.

4. **Wire the relay**:
   Connect relay NO contacts across the boiler's thermostat input. Test by toggling `switch.thermostat_boiler_relay` from Home Assistant and confirming the boiler fires.

5. **Add automations and dashboard**:
   Copy the YAML from `automations.yaml` and `dashboard.yaml` into Home Assistant.

## Configuration

### Setpoints

Adjust substitutions in `smart-thermostat-relay.yaml`:

| Substitution | Default | Meaning |
|--------------|---------|---------|
| `comfort_temp` | 21.0 | Comfort preset target |
| `eco_temp` | 17.0 | Eco preset target |
| `max_safe_temp` | 28.0 | Hard relay shutoff |

### Hysteresis and Cycle Times

In the `climate:` block:

- `heat_overrun: 0.5` and `heat_deadband: 0.5` give a 1.0 C total deadband. Lower values mean tighter control but more cycling.
- `min_heating_off_time: 15min` is the anti short-cycle window. Increase to 20-30 min for older oil/gas boilers; reduce for electric heaters.
- `min_heating_run_time: 5min` keeps each heating call long enough to do real work.

## Safety Notes

1. **Boiler short-cycling damages equipment**. Do not lower `min_heating_off_time` below 10 min for gas/oil boilers.
2. **Hard cutoff is defensive**, not primary. The ESPHome safety cutoff at 28 C is a backup against runaway control logic; do not rely on it as your overheat protection. A proper limit thermostat in the boiler itself is the real safeguard.
3. **Manual override**: Toggling `switch.thermostat_boiler_relay` while the thermostat is also calling for heat will fight the climate logic. Use it only for installation testing.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Temperature reads -127 C | DS18B20 wiring or missing 4.7k pullup |
| Boiler short-cycles | Increase `min_heating_off_time`, widen deadband |
| Setpoint never reached | Heater undersized, or probe in cold draft |
| Probe reads 2-3 C high | Probe too close to ESP32 or relay; move it on a longer wire |
| Relay clicks but boiler ignores | Check NO vs NC contacts, verify boiler input voltage |
